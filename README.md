Collection of Containers and Algorithms.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  crystalline:
    github: jtomschroeder/crystalline
```

## Content

- Algorithms
  - KMP-Search
  - Search
- Containers
  - Heap
  - KD-Tree
  - Queue
  - Priority Queue
  - RB Tree Map
  - Splay Tree Map
  - Stack
  - Suffix Array
  - Trie
- Graph
